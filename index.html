My Projects

Project 1: Educational Blog Website

A website created to summarize and share lecture materials from my courses. Built using HTML, CSS, Bootstrap, and JavaScript, this blog allows users to easily navigate through various topics and access summarized educational content in a simple and interactive format.

Project 2: Daily Activity Website

During the month of Ramadan, I developed a website to document and share my daily activities through video. Created with HTML, CSS, Bootstrap, and JavaScript, the website showcases a personal log of my activities throughout the day, serving both as a project and a means to engage with daily routines during this special month.

Project 3: Finance Management Website

This website enables users to manage their finances by tracking income and expenses. Developed with HTML, CSS, Bootstrap, JavaScript, and PHP, the site features user registration, login, and a virtual account system where users can transfer funds between each other. It is designed for individuals who want to better manage their financial activities through an intuitive platform.

Project 4: Obesifit Educational Website

A platform to provide educational content about obesity, where users can access articles, videos, and tools like a BMI calculator and daily calorie counter. The website was built with HTML, CSS, JavaScript, React, and PHP, and includes a consultation feature for users to interact with healthcare professionals. Doctors can also upload content to share with users. This site aims to raise awareness and provide valuable resources for managing obesity.

Project 5: Mosque Administration System

A desktop application for mosque administration built with C#. The application helps manage mosque income and expenses, activity scheduling, inventory, and calculates Zakat. Additionally, it includes a Qibla direction feature and an AI chatbot for mosque-goers to ask questions. The application simplifies the mosque management process and enhances administrative efficiency.
